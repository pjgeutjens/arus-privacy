Arus Privacy Policy

Arus is a local Android app for tracking selected bank transaction notifications. The app is designed to keep financial data on the user's device unless the user explicitly chooses to export or share it.

Effective date: July 9, 2026

Arus does not run a cloud service, does not sell personal data, and does not automatically upload transaction or notification data to Leap Consulting or any third party.

Data Arus Processes

After the user grants Android notification access, Arus can read notifications from supported banking apps. Depending on what the bank includes in the notification, this may include:

  • Bank app name and package name.
  • Notification title, message text, expanded text, and notification timestamp.
  • Parsed transaction details such as amount, merchant or sender, direction, bank, account label, and transaction time.
  • Account aliases, notes, manually entered balance checkpoints, calculated balances, and app diagnostic metadata.

How Data Is Used

Arus uses captured notification data to build a local transaction timeline, estimate account balances from user-confirmed balance checkpoints, detect unsupported notification formats, and create user-requested diagnostic exports.

Storage And Security

Arus stores app data locally on the Android device. The local database is encrypted at rest using SQLCipher. Access to notification data requires the Android notification listener permission, which the user can enable or disable in Android system settings.

Diagnostics And Sharing

Arus can create a diagnostics report to help improve notification parsing. This report may include raw notification text and parsed transaction data. The report is created only when the user taps the diagnostics export action and chooses where to send it through Android's share or email interface.

Arus does not automatically send diagnostics reports. The user controls whether a report is shared and who receives it.

Third Parties

Arus does not include advertising SDKs, analytics SDKs, or third-party tracking. If the user chooses to send a diagnostics report by email or another sharing app, that chosen app or service processes the shared file under its own privacy practices.

Data Retention And Deletion

Arus keeps local transaction, notification, and balance data until the user deletes it, wipes data inside the app, uninstalls the app, or clears app storage through Android settings.

User Controls

  • The user can disable notification access in Android settings at any time.
  • The user can delete local app data using the app's wipe action or Android system storage controls.
  • The user can choose whether to export diagnostics or CSV files.

Children

Arus is not intended for children and is not designed to collect data from children.

Changes

This policy may be updated when Arus changes how it processes or protects data. The effective date above will be updated when material changes are made.

Contact

For privacy questions, contact the developer using the support email address listed for Arus on Google Play.
